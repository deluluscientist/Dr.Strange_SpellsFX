# Dr.Strange_SpellsFX 

A browser-based Augmented Reality (AR) experience that turns your hand gestures into mystic spells. This project uses **MediaPipe Hands** for computer vision and **HTML5 Canvas** for high-performance visual effects.

---

## ⚡ Features
* **Real-time Hand Tracking:** Detects 21 hand landmarks with low latency.
* **Gesture Recognition:** Custom logic for detecting complex hand shapes.
* **Mystic Visuals:** Dynamic canvas animations including particles, kaleidoscopes, and mathematical fractals.
* **Immersive HUD:** A "Cinzel" styled interface with a responsive gesture guide.

## 🛠️ Technology Stack
* **AI Engine:** [MediaPipe Hands](https://google.github.io/mediapipe/solutions/hands)
* **Rendering:** HTML5 Canvas API (2D Context)
* **Typography:** Google Fonts (Cinzel Decorative)
* **Language:** Vanilla JavaScript (ES6+)
